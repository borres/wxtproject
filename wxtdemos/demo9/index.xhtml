Describing the Bezier function.

This is a demonstration of a some document manipulation. The basic source is the file Bezier.odt. The link is here:Bezier.odt. If you have Open Office installed, there is a chance your browser will reckognize it and suggest that it is opened in Open Office Document Writer.

I do the following experiments with this file, Bezier.odt.

  1. I export the file from Open Office as xhtml, see BezierAuto.xhtml
  2. I save the file from Open Office as pdf, see BezierAuto.pdf
  3. I use WXT to extract pieces from the file, add some stuff, and build a web site, see link Bezier in left menu. I also let WXT collect from the website, collected.xhtml and I let WXT apply Prince to build PDFcollected.pdf.
    The script for this site is like this:
    _script.xml

I also add a few applets to the web modules, and I have written some stuff in HTML in an accessible editor, like Notepad++ or Adobe Dreamweaver. The applets are directly accessible from the net.

Please note that this is neither a demonstration of pedagogy nor mathematics, it is just a demonstration of some of the features in WXT.

Do also note there are some restrictions on WXT's ability to handle OO-format. Essentially it is restricted to headers, paragraphs, tables, images, formulas and lists. WXT does not replicate styling.
