Welcome

We will do a few modifications to the set up in demo A. Firstly we will prepare for more templates. We do this by defining header and footer as fragments and put them on a file, gfx/myfragments.xml. This will simplify the template(s), and make it easy to work on the details both in the fragments and the template(s).

A simple template:

+
-
<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "-//W3C//DTD XHTML 1.0 Strict//EN">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" lang="en">
<head>
  <title><?_wxt fragment id="_name"?></title>
  <meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
  <meta name="GENERATOR" content="WXT"/>
  <meta name="KEYWORDS" xmlcontent="WXT,XML,WebSite"/>
  <link rel="STYLESHEET" href="template.css"/>
  <link rel="STYLESHEET" href="wxtstyle.css"/>
</head>

<body>

<!-- header -->
<?_wxt fragment id="header"?>
<!-- header -->

<!-- left menu -->
<div class="menu" id="menu">
  <?_wxt modulemenu books="all"?>
</div><!-- left menu -->

<!-- content -->
<div class="content">
  <div class="collectable">
    <div class="main">
      <?_wxt importxml xpath="//div[@class='mainblock']/*"?>
    </div>
  </div>
</div><!-- content -->


<!-- footer -->
<?_wxt fragment id="footer"?>
<!-- footer -->


</body>
</html>
